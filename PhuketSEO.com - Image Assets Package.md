## 📌 คำแนะนำสำหรับ Cursor AI
1. นำรูปภาพไปวางใน `/public/images/` หรือ `/assets/images/` ตาม project structure
2. ใช้ Alt Text ตามที่แนะนำในตารางด้านบน
3. แปลงเป็น WebP format เพื่อ performance ที่ดีขึ้น (ใช้ `sharp` หรือ `next/image`)
4. ใช้ `loading="lazy"` สำหรับรูปที่อยู่ below the fold
5. ใช้ `priority` หรือ `loading="eager"` สำหรับ Hero Images เท่านั้น
