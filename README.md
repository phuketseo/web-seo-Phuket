# PhuketSEO — Next.js App Router

เว็บไซต์ Digital Marketing Agency สำหรับภูเก็ต สร้างด้วย Next.js 15 App Router + Tailwind CSS 4

## เริ่มต้นใช้งาน

```bash
npm install
npm run dev
```

เปิด [http://localhost:3000](http://localhost:3000) ในเบราว์เซอร์

## Build & Deploy

```bash
npm run build
npx vercel --prod
```
